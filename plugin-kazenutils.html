Descripción

KazenUtils es una colección de utilidades y herramientas diseñadas para facilitar la administración del servidor y mejorar la experiencia de juego. Este plugin todo-en-uno contiene múltiples funcionalidades que normalmente requerirían varios plugins diferentes.

Características

  • Sistema de teletransporte avanzado
  • Gestión de inventarios virtuales
  • Comandos de administración simplificados
  • Sistema de mensajes privados mejorado
  • Herramientas de moderación para el servidor
  • Sistema de logs detallados
  • Control de clima y tiempo
  • Más de 50 comandos útiles para administradores y jugadores

Cómo usar

Instalación:

  1. Descarga el archivo .phar del plugin
  2. Colócalo en la carpeta "plugins" de tu servidor
  3. Reinicia el servidor
  4. Edita el archivo de configuración según tus preferencias

Comandos principales:

/ku help
Muestra todos los comandos disponibles
/warp [nombre]
Teletransporta al jugador a un warp específico
/invsee [jugador]
Permite ver y editar el inventario de otro jugador
/ku reload
Recarga la configuración del plugin

Configuración

KazenUtils tiene un sistema de configuración modular. Puedes habilitar o deshabilitar cada componente individualmente según las necesidades de tu servidor. 

# KazenUtils - Configuración principal
modules:
  teleport: true
  inventory: true
  moderation: true
  chat: true
  economy: false
  weather: true

permissions:
  use-admin-commands: "kazenutils.admin"
  bypass-cooldowns: "kazenutils.bypass"
