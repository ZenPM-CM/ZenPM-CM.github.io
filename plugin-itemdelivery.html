Descripción

ItemDelivery es un plugin potente y flexible que permite a los administradores del servidor configurar sistemas de entrega de items a los jugadores mediante comandos simples. Ideal para servidores de supervivencia, minijuegos o cualquier otro tipo de servidor que necesite un sistema de gestión de items.

Características

  • Sistema de entrega de items configurable
  • Soporte para permisos por grupos
  • Múltiples configuraciones para diferentes sets de items
  • Comandos intuitivos para administradores y jugadores
  • Registro detallado de todas las entregas
  • Compatible con PlaceholderAPI
  • Mensajes personalizables

Cómo usar

Instalación:

  1. Descarga el archivo .phar del plugin
  2. Colócalo en la carpeta "plugins" de tu servidor
  3. Reinicia el servidor
  4. Configura el archivo config.yml según tus necesidades

Comandos:

/itemdelivery give [player] [kit]
Entrega un kit específico a un jugador
/itemdelivery list
Muestra todos los kits disponibles
/itemdelivery reload
Recarga la configuración del plugin

Configuración:

# ItemDelivery config.yml
settings:
  language: es
  debug: false

kits:
  starter:
    items:
      - "STONE_SWORD:1:0"
      - "BREAD:16:0"
    cooldown: 86400 # 24 horas en segundos
    permission: "itemdelivery.kit.starter"
  vip:
    items:
      - "DIAMOND_SWORD:1:0"
      - "GOLDEN_APPLE:8:0"
    cooldown: 43200 # 12 horas en segundos
    permission: "itemdelivery.kit.vip"
