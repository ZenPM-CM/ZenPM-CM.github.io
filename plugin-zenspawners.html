Descripción

ZenSpawners es un sistema avanzado de gestión de spawners que permite a los jugadores personalizar y mejorar sus generadores de mobs. Con características únicas como niveles, habilidades especiales y un sistema de estadísticas, este plugin transformará la experiencia de granjas en tu servidor.

Características

  • Sistema de niveles para spawners (hasta 10 niveles)
  • Habilidades desbloqueables para cada tipo de spawner
  • Menú GUI intuitivo para gestionar tus spawners
  • Estadísticas detalladas de cada spawner
  • Integración con economía (Vault)
  • Sistema de logros relacionados con spawners
  • Soporte para stacks de spawners
  • API para desarrolladores

Cómo usar

Instalación:

  1. Descarga el archivo .phar del plugin
  2. Colócalo en la carpeta "plugins" de tu servidor
  3. Asegúrate de tener Vault instalado
  4. Reinicia el servidor
  5. La configuración se generará automáticamente

Comandos:

/zenspawners
Abre el menú principal de ZenSpawners
/zenspawners give [player] [type]
Da un spawner específico a un jugador
/zenspawners upgrade
Mejora el spawner que estás mirando
